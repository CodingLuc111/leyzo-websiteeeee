⚙️ Einrichtung erforderlich
Bevor die Website funktioniert, musst du deine Supabase-Daten in der index.html eintragen.
1
Supabase Projekt öffnen
Gehe auf supabase.com → dein Projekt → Settings → API
2
SUPABASE_URL ersetzen
Kopiere die Project URL und ersetze in der index.html:
const SUPABASE_URL = 'https://DEIN-PROJEKT.supabase.co';
3
SUPABASE_ANON_KEY ersetzen
Kopiere den anon public Key und ersetze:
const SUPABASE_ANON = 'eyJhbGci...';
Dann Datei speichern und neu laden. 🚀
